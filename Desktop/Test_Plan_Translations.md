# Translations

## Precondition

Change system language on Mac OS

1.  Click on Apple menu in topleft corner
2.  Click on System Preferences
3.  Select Language and Region
4.  Add the requested language into Preferred languages
5.  Change it to your primary language
6.  Restart your computer
